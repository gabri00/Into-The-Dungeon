# Java fight game
 Explore a dungeon from your terminal!

## Usage
`java Main <options>`

#### Options
- `-h` or `--help` to display the game instructions
- `-l` or `--load` to load a previous saved game

### Enemies
- Slime
- Goblin
- Ogre
- Demon
- Archdemon
- Dragon

### Weapons
- Bow
- Sword
- Spear

### Artifacts
- Necklace
- Armlet
- Ring
